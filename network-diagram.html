Perimeter firewall requirement (rows 3–4): the hotel firewall must allow 10.0.0.10 → ElevenLabs (199.88.252.34) TCP 5061 and the SRTP media UDP range, and have SIP ALG disabled. All other outbound SIP stays blocked.
NAT note: the middleware is private 10.0.0.10, presented to the internet as 203.0.113.10 via the hotel gateway. Asterisk advertises 203.0.113.10 for media so ElevenLabs returns audio correctly. If the public IP changes, the Asterisk external_media_address must be updated.
